# Smart Rural Governance & Emergency Management System

A MERN-based application for rural governance and emergency response workflows.

## Project Structure

- `client/` - React + Vite frontend
- `server/` - Express + MongoDB backend

## Features

- Citizen login and registration
- Complaint filing and tracking
- Certificate application and verification
- Panchayat and district officer dashboards
- Emergency SOS and volunteer coordination
- Real-time updates through Socket.IO

## Requirements

- Node.js
- MongoDB

## Setup

### Backend

```bash
cd server
npm install
npm run dev
```

### Frontend

```bash
cd client
npm install
npm run dev
```

## Notes

- Configure environment variables in the `server/` folder before starting the backend.
- Run the frontend and backend in separate terminals.
